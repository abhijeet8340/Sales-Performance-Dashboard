📊 Sales Performance Dashboard Project
📌 Overview

Welcome to my Sales Performance Dashboard Project!
This data analytics project focuses on cleaning, analyzing, and visualizing sales data using Microsoft Excel. The primary objective is to build an interactive dashboard that provides meaningful business insights and helps track sales performance effectively.

The dashboard transforms raw sales records into structured reports using KPIs, charts, and interactive filters.

📖 Project Description

Sales analysis plays a crucial role in evaluating business performance and identifying growth opportunities. In this project, I used Microsoft Excel to clean and preprocess structured sales data, perform exploratory data analysis (EDA), and design a dynamic dashboard.

The dashboard highlights important Key Performance Indicators (KPIs) such as:

Total Sales Revenue

Sales by State

Product Performance

Monthly Sales Trends

Category-wise Contribution

Target vs Achievement

The goal of this project is to demonstrate my skills in data cleaning, KPI tracking, dashboard design, and business insight generation required for a Data Analyst role.

📂 Dataset

The dataset includes the following fields:

Order Date

State

Product Name

Category

Sales Amount

Quantity

Target

Profit

📊 Dashboard Highlights
🔹 Total Sales KPI

Displays overall revenue performance with summary metrics.

🔹 Sales by State – Map Chart

Shows regional sales distribution to identify high-performing and underperforming states.

🔹 Product Analytics – Column Chart

Analyzes sales contribution by product and category.

🔹 Monthly Analytics – Line/Area Chart

Tracks monthly sales trends to identify seasonal patterns.

🔹 Interactive Filters (Slicers)

Allows dynamic filtering by state, product, and date.

🛠 Tools Used

Microsoft Excel

Pivot Tables & Pivot Charts

Advanced Excel Formulas (SUMIFS, IF, COUNTIFS)

Conditional Formatting

KPI Tracking Techniques

🚀 How to Use

Download the Excel file from the repository.

Open it in Microsoft Excel.

Use slicers and filters to interact with the dashboard.

Analyze trends and performance metrics dynamically.
